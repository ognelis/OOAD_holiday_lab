Реализовано на scala.

Классы и программа находятся в ./src/main/scala/

Данная система описывает процесс снятие денег с банкомата.
Так как сумма которую мы хотим снять проходит по цепочке обработчиков,
деннежных кассет с определенным номиналом. Каждая кассета отзывается на
события и если сможеть дать наличность, то даст, и передает управления следующей
выбранной нами кассете.

Система довольна упрощена, она не имеет специальных классов, объектов, для работы
с денежными кассетами, банкомат работает с ними напрямую, что затрудняет
выбор одиннаковых по номиналу денежных кассет для операций, нужен отдельный объект
